Abstract We consider different models for intraday log-returns: Lévy models, symmetric models, and Lévy processes subjected to independent continuous time-changes. For these models, we show bivariate interchangeability of intraday up- and downside volatility ratios which are built using daily high-low prices. Using conditional inference permutation tests on bivariate interchangeability, we develop an omnibus test for the above-mentioned models.
