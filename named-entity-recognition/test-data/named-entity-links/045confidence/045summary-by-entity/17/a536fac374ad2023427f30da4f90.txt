Abstract The following result is proved: Letp>0,a>−1. Suppose thatG is a measurable subset ofB, the unit ball in ℝ N , for which there exists a positive constantA 1, so that $$\\int\\limits_B {\\left( {1 - \\left| x \\right|} \\right)^a \\left| {f(x)} \\right|^p dm \\leqslant A_1 } \\int\\limits_G {\\left( {1 - \\left| x \\right|} \\right)^a \\left| {f(x)} \\right|^p dm}$$ for each function that is harmonic inB and for which the left-hand side of the above inequality is finite. Then there is a positive constantA 2 so that for each ballK with center on ϖB, $$m\\left( {K \\cap B} \\right) \\leqslant A_2 m\\left( {K \\cap G} \\right).$$ Herem denotes Lebesgue measure in ℝ N . This result answers a question left open byDan Luecking [2].
